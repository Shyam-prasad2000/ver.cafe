# Ver.Cafe - Roots

Thrissur’s modern specialty coffee sanctuary. This repository contains the frontend web application and the Sanity Headless CMS configuration for Ver.Cafe - Roots.

## Tech Stack
- **Frontend**: React, Vite, Lucide Icons, Framer Motion
- **CMS**: Sanity (v3+)
- **Server**: Express.js
